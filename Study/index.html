Convolutional Neural Network (CNN)

The operation process of a Convolutional Neural Network (CNN) mainly involves the following steps:

  1. The input layer is responsible for image input.
  2. Then, through the convolution layer, features of the image are extracted. The convolution layer performs convolution operations on the data using a filter/kernel, and through sliding operations, features of the entire image are extracted to generate corresponding feature maps.
  3. Using max pooling or average pooling, downsampling is performed, reducing the size of the feature maps while retaining important features.
  4. The pooled feature maps are passed to a neural network, which predicts the image content.

Recurrent Neural Network (RNN)

The main feature of a Recurrent Neural Network (RNN) is its ability to handle sequential or time-dependent data. Due to its recurrent connections and unfolding, each output depends not only on the current input but also on the hidden state from the previous time step.

However, when dealing with long sequences, due to the unfolding nature, the network may encounter the problem of vanishing/exploding gradients. As shown in the diagram, when there are 50 previous data points, repeated multiplication by (w_2) causes the gradient to shrink to (2^{-50}), leading to the vanishing gradient problem. Similarly, if (w_2) is greater than 1, it leads to exploding gradients.

Long Short-Term Memory (LSTM)

Long Short-Term Memory (LSTM) effectively addresses the vanishing/exploding gradient problem in RNNs. The architecture is shown in the diagram above.

  • The green line at the top represents long-term memory (L), also known as the cell state, which is used to store previously inputted data.
  • The blue section represents the forget gate, which processes the current input through a sigmoid function to generate a corresponding forget rate. This value is multiplied by the long-term memory (L) to indicate how much to “remember.”
    • If the calculation results in 97%, most of the long-term memory will be retained.
    • If the calculation results in 5%, most of the long-term memory will be forgotten.
  • The green and yellow sections represent the input gate. The yellow section computes the potential long-term memory, used to generate the content that the long-term memory (L) needs to remember. The green section computes how much of the potential long-term memory to remember (% Potential Memory To Remember). The two values are multiplied and added to the long-term memory (L).
  • The purple and pink sections represent the output gate. First, it computes how much of the short-term memory to remember (% Potential Memory To Remember), using the input and short-term memory. Then, similar to the long-term memory, this value is used to generate the final prediction.
